## 本製作物について
本製作物はReact、TypeScript、Tailwind CSSを使用して開発した**付箋**をモチーフにしたTodoアプリです。アナログでタスク管理をするときは自分は付箋をよく使うので、それをデジタルでも実装してみたいと思い製作に至りました。<br>
機能面は一般的なTodoアプリと同じでタスク名、優先度、期限、詳細の四項目をタスクとして管理可能になっています。今回特に私が挑戦した部分はデザイン・UI面です。まず、下に表示されている画像を比較するとわかる通り、タスクの追加をする部分はハンバーガーメニューとしました。これは、画面いっぱいにタスクを配置できるようにするためと、アスペクト比が異なるデバイスであっても使いやすいようにするためです。次に、二つ目と三つ目の画像の通り、タスクの配置を自由自在に配置できるようにしました。イメージとしてはノートに付箋を貼っていくような雰囲気です。これが本製作物の最大の特徴です。優先度に応じて付箋の色が変わるようにもしました。
